## What is KozoFlow?

KozoFlow is a unified AI platform designed to optimize **Hiring** and **Software Quality** at Kozocom. It combines:

- **Talent Scanner** - AI-powered CV parsing and candidate matching
- **Code Review Analytics** - Track and visualize code quality metrics
- **Smart Integrator** - Central AI chatbot connecting all data sources
